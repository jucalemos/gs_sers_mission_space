# gs_sers_mission_space

Projeto desenvolvido para a Global Solution da disciplina de Soluções em Energias Renováveis e Sustentáveis.

## Sobre o Projeto

O Acqua Vita é um sistema desenvolvido em Python para simular o monitoramento de condições oceânicas utilizando dados inspirados em tecnologias de observação espacial.

A aplicação analisa informações como temperatura da água, altura das marés, presença de manchas de óleo, acúmulo de plástico e emissões ilegais. Com base nesses dados, o sistema gera alertas e recomendações para auxiliar na preservação dos oceanos e das regiões costeiras.

## Objetivo

O objetivo do projeto é criar um sistema capaz de:

* Monitorar condições ambientais oceânicas;
* Identificar situações de poluição marítima;
* Detectar derramamentos de petróleo;
* Gerar alertas automáticos;
* Auxiliar na tomada de decisões.

## Funcionalidades

* Simulação de dados ambientais;
* Monitoramento das condições oceânicas;
* Geração automática de alertas;
* Recomendações inteligentes baseadas nos dados analisados;
* Exibição dos resultados no terminal.

## Tecnologias Utilizadas

* Python
* Biblioteca Random
* Estruturas condicionais
* Lógica de programação

## Conclusão

O Acqua Vita demonstra como a programação pode ser utilizada para monitorar condições ambientais e identificar riscos de forma simples e eficiente. A solução une tecnologia e sustentabilidade para auxiliar na proteção dos oceanos e das regiões costeiras.
